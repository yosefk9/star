تگ strong تگی است که با استفاده از آن میتوان یک متن / محتوا را بصورت مهم یا متنی که دارای ارزش محتوایی می باشد نشان داد. تگ B و تگ Strong در ظاهر دقیقا یکی هستن، یعنی هردوتاشون متن رو بصورت پرُرنگ ( Bold ) نشون میدن ،ولی خب از نظر مرورگرها و موتورهای جستجوگر باهمدیگه کاملا فرق میکنند. تگ Strong : یک متن با ارزش محتوایی و مهم ایجاد میکند ( پس موتورهای جستجوگر بهش توجه میکنند ) تگ B : هیچ ارزش محتوایی نداره و فقط متن رو پرُرنگ ( Bold ) میکنه
